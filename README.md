## ⚡ Sobre el Proyecto

**AUTOMACER** es el rostro digital de una empresa argentina líder, fundada en 1991, especializada en servicios industriales eléctricos de alta precisión. Este proyecto web no es solo un portal corporativo; es una declaración de intenciones. Combina un diseño **dark-tech** vanguardista con una arquitectura frontend de altísimo rendimiento para reflejar la confiabilidad, modernidad y seguridad que la empresa ofrece en cada una de sus instalaciones y reparaciones críticas.

---

## 🛠️ Servicios Principales

Nuestro ecosistema digital exhibe de manera interactiva la amplia gama de soluciones industriales de AUTOMACER:

- ⚙️ **Programación y reparación de PLC**: Automatización inteligente y controladores lógicos.
- ☀️ **Instalaciones Fotovoltaicas**: Energía renovable y sustentable para el consumo industrial.
- 🔧 **Mantenimiento Preventivo y Predictivo**: Anticipación de fallas para mantener las líneas de producción operativas.
- 🚀 **Reparaciones Críticas**: Asistencia técnica en tiempo récord para emergencias industriales.
- 🏭 **Instalación de Sistemas Industriales**: Puesta en marcha integral y segura de infraestructuras complejas.

---

## 🎨 Identidad Visual y Experiencia de Usuario (UX)

El sitio fue diseñado bajo un estricto concepto de **Glassmorphism** y estilo industrial moderno:

* **Paleta de Colores Premium**: Un fondo profundo y elegante (`Surface: #0e1322`) que contrasta magistralmente con acentos vibrantes (`Cyan Brand: #00D4FF`), transmitiendo energía, innovación y precisión técnica.
* **Micro-interacciones Dinámicas**: Impulsado por **Framer Motion**, el sitio cobra vida con animaciones sutiles, efectos *glow* en botones de llamado a la acción (CTAs) y transiciones fluidas que elevan la percepción de la marca.
* **Rendimiento Optimizado**: Uso de `react-intersection-observer` para animar elementos únicamente al hacer scroll (lazy animation), garantizando un rendimiento impecable y una navegación sin interrupciones.

---

## 💻 Arquitectura Tecnológica

Este proyecto está construido con las mejores tecnologías modernas de frontend, priorizando la escalabilidad, la velocidad y una experiencia de desarrollo superior:

- **React 19**: Componentes modulares, estado eficiente y las últimas características del ecosistema React.
- **Vite**: Entorno de desarrollo ultra rápido con HMR (Hot Module Replacement) y construcciones optimizadas.
- **Tailwind CSS v4**: Utilidades de diseño ágiles para maquetar interfaces complejas, adaptativas y con estilo *Dark Mode* nativo.
- **Framer Motion**: Librería de animación declarativa de grado de producción.
- **Zod & DOMPurify**: Validación robusta de datos y sanitización para máxima seguridad en la interacción de los usuarios.

## 🚀 Instalación y Desarrollo Local

Si deseas correr el proyecto en tu entorno local, sigue estos pasos:

1. **Clonar el repositorio y acceder al directorio:**
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   cd automacer
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```
